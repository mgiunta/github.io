Analysis of BeerAdvocate Review Data¶

Description: "This dataset consists of beer reviews from Beeradvocate. The data span a period of more than 10 years, including all ~1.5 million reviews up to November 2011. Each review includes ratings in terms of five "aspects": appearance, aroma, palate, taste, and overall impression. Reviews include product and user information, followed by each of these five ratings, and a plaintext review. We also have reviews from ratebeer."

Some questions of interest:

1.  Which brewery produces the strongest beers by ABV%?  
2.  If you had to pick 3 beers to recommend using only this data set, which would you pick?
3.  Which factors (e.g. aroma, taste, appearance, palate) are most important in determining the overall quality of a beer?
4.  If I typically enjoy a beer due to its aroma and appearance; which beer style should I try?

Q2. If you had to pick 3 beers to recommend using only this data set, which would you pick?¶

In [10]:
#My recommendations are based on the product of popularity (normalized total number of reviews) and overall ratings.
#   Essentially, this is a crowd-source solution with the rationale that it's more difficult to uphold a high rating as the
#   number of reviews increases (or as time goes on).  The problem here is that smaller breweries making excellent beers
#   but with weaker popularity/distribution get penalized.  However, popularity is a function of consumer demand, which is
#   driven in part by quality and taste.

#Q2.1 Which beers are most popular based on total number of reviews?
df_pop_beers = df.groupby(['brewery_name2','beer_name2']).size().reset_index(name='review_count')
df_pop_beers = df_pop_beers.sort_values(by=['review_count'], ascending=False).reset_index()

#Q2.2 Which beers have the highest average overall review ratings?
df_best_beers = df.groupby(['beer_name2'])[['review_overall']].mean()
df_best_beers.reindex(columns=['review_overall'])
df_best_beers = df_best_beers.sort_values('review_overall', ascending=False).reset_index()

#Merge popular beers with best reviewed beers
df2 = pd.merge(df_pop_beers, df_best_beers, on='beer_name2', how='inner')

#Normalize the review count column, get 
df2['normalized_review_count'] = (df2['review_count']-df2['review_count'].min())/(df2['review_count'].max()-df2['review_count'].min())
df2['popular_review_score'] = df2['normalized_review_count'] * df2['review_overall']
df2 = df2.sort_values(['popular_review_score'], ascending=False).reset_index()

print('My #1 recommended beer is: ' + str(df2.beer_name2[0]) + ' by: ' + str(df_pop_beers.brewery_name2[0]) )
print('My #2 recommended beer is: ' + str(df2.beer_name2[1]) + ' by: ' + str(df_pop_beers.brewery_name2[1]) )
print('My #3 recommended beer is: ' + str(df2.beer_name2[2]) + ' by: ' + str(df_pop_beers.brewery_name2[2]) )

Q4. I typically enjoy a beer due to its aroma and appearance; which beer style should I try?¶

In [12]:
#Determine mean review_aroma and review_appearance for each beer_style, then retrieve beer_style with maximum value

df_style1 = df.groupby(['beer_style'])[['review_aroma']].mean()
df_style2 = df.groupby(['beer_style'])[['review_appearance']].mean()

df_style1.reindex(columns=['review_aroma'])
df_style1 = df_style1.sort_values('review_aroma', ascending=False).reset_index()

df_style2.reindex(columns=['review_appearance'])
df_style2 = df_style2.sort_values('review_appearance', ascending=False).reset_index()

#print(df_style1.head())
#print(df_style2.head())

df4 = pd.merge(df_style1, df_style2, on='beer_style', how='inner')

df4['aroma_app_score'] = df4['review_aroma'] * df4['review_appearance']
df4 = df4.sort_values(['aroma_app_score'], ascending=False).reset_index()

print('A4: The best beer style to try for maximum aroma and appearance is: ' + str(df4.beer_style[0]))

A4: The best beer style to try for maximum aroma and appearance is: American Double / Imperial Stout
